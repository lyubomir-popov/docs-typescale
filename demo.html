Vanilla Framework Integration

This demo uses vanilla framework with custom overrides:

  • Responsive font sizes using calc()
  • Baseline-aligned typography (h1-h6 and p automatically aligned)
  • Ubuntu Sans font family
  • Precise baseline grid alignment

Usage

To use this in your project:

  1. Include dist/main.css in your HTML
  2. Use .text-responsive for responsive text
  3. All h1-h6 and p elements are automatically baseline-aligned
  4. Add .baseline-grid to containers for visual alignment

Configuration

Edit config/typography-config.json to customize:

  • Font sizes and line heights
  • Baseline unit (currently 0.5rem)
  • Font family and file

The watcher will automatically regenerate tokens and update the CSS.
